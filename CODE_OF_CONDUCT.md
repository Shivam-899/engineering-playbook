> Version: 1.0.0
>
> Status: Stable
>
> Last Updated: 2026-06-30

---

# Our Commitment

The Engineering Playbook is built to encourage respectful, constructive, and professional collaboration.

We welcome contributions from engineers of all experience levels.

Everyone should feel comfortable asking questions, sharing ideas, reporting issues, and proposing improvements.

---

# Expected Behavior

Contributors are expected to:

- be respectful
- communicate professionally
- provide constructive feedback
- focus on technical discussions
- welcome different viewpoints
- help others learn

---

# Unacceptable Behavior

The following behaviors are not acceptable:

- harassment
- discrimination
- personal attacks
- abusive language
- trolling
- intimidation
- publishing private information
- intentionally disruptive behavior

---

# Engineering Discussions

Technical disagreements are expected.

Discuss ideas using:

- evidence
- benchmarks
- documentation
- engineering reasoning

Avoid personal arguments.

---

# Enforcement

Repository maintainers may:

- request changes
- remove inappropriate content
- close discussions
- reject contributions
- restrict participation when necessary

The goal is always to maintain a productive engineering community.

---

# Scope

This Code of Conduct applies to:

- Issues
- Pull Requests
- Discussions
- Documentation
- Code Reviews
- Community interactions related to this repository

---

# Reporting

If you encounter behavior that violates this Code of Conduct:

- contact the maintainers
- provide relevant information
- allow maintainers to investigate before drawing conclusions

Reports will be handled respectfully and confidentially whenever practical.
